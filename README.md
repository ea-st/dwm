Software dependencies
---------------------

- [Alacritty](https://github.com/alacritty/alacritty)

- libxcb (required by swallow)
- Xlib-libxcb (required by swallow)
- xcb-res (required by swallow)

Installation
------------


    git clone https://github.com/ea-st/dwm;cd dwm

 ------------------------------------------------

    make install
